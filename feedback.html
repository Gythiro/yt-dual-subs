GitHub Issues — trackable

GitHub Issues——可追踪进度

If you have a GitHub account, an issue is the best way to report bugs: there are guided templates, you'll see progress, and get notified when it's fixed.

有 GitHub 账号的话，提 issue 是报 bug 的最佳方式：有引导模板、能看到处理进度、修好了还会收到通知。

Store reviews — for the love

商店评论——留个好评

Reviews keep an independent open-source project alive — but they're a poor place to debug: replies are slow and back-and-forth barely works. Love it? Leave a review. Something broken? Email is much faster.

评分是独立开源项目活下去的氧气——但评论区不适合排查问题：回复慢、来回沟通很困难。喜欢就去评个分；出问题了，发邮箱快得多。
